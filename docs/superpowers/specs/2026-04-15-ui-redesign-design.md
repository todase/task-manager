# UI Redesign — Design Spec

**Дата:** 2026-04-15  
**Подход к реализации:** итеративный рефакторинг (компоненты по одному, без изменения структуры данных)

---

## 1. Визуальный стиль

**Направление:** Clean / Light — белые карточки на тёплом фоне, скруглённые углы, мягкие тени, иконки вместо текстовых кнопок.

## 2. Навигация (верхняя часть)

### Фильтр по дате — Segmented control

Четыре сегмента всегда видны: **Сегодня · Неделя · Все · Потом**  
Стиль: iOS-подобный переключатель — фон `bg-gray-200`, активный сегмент `bg-white shadow-sm`.

### Проекты — аккордеон

Свёрнутая строка с иконкой папки, подписью «Проекты» и счётчиком.  
При раскрытии — таблетки проектов + кнопка «+ проект».  
Клик на проект — выбирает его и сворачивает аккордеон.  
Состояние (открыт/закрыт) — локальный `useState`.

### Теги — аккордеон

Аналогично проектам: свёрнутая строка с иконкой тега и счётчиком.  
При раскрытии — цветные таблетки тегов + кнопка сброса если есть активные.  
Клик на тег — переключает его (мульти-выбор), аккордеон не сворачивается.

## 3. Список задач (TaskItem)

### Свёрнутое состояние (всегда)

Одна строка, только три элемента:
- Чекбокс (круглый, `border-2 rounded-full`)
- Название задачи (`text-sm font-medium`)
- Бейдж даты (цветной пилл — серый / синий / красный / зелёный)

Никаких тегов, иконок повторения, счётчиков подзадач, описания и кнопок в строке.

### Раскрытое состояние (по клику на строку)

Строка разворачивается вниз с синей левой границей (`border-l-2 border-blue-500`).  
Появляется:
- Строка тегов (цветные пилы)
- Иконка повторения + текст (`RefreshCw · еженедельно`)
- Описание (текст курсивом, редактируется по клику)
- Секция подзадач с чекбоксами и счётчиком
- Кнопка «Удалить» (`Trash2` + текст, красная)

Стрелка `ChevronUp` в правом углу заголовка строки.

## 4. Форма добавления задачи — модальное окно

Кнопка добавления — FAB (floating action button), `position: fixed`, внизу справа.  
На мобильном: синий круг 48×48px с `+`, `shadow-lg`.  
На десктопе: та же FAB-кнопка, `position: fixed`, внизу справа.

По нажатию открывается модальное окно:
- Мобильный: bottom sheet (выезжает снизу, ручка наверху)
- Десктопе: центрированный модал с backdrop

Содержимое модала:
1. Поле ввода названия (автофокус)
2. Кнопки-иконки: 📅 Дата · ↻ Повтор · 🏷 Метка
3. Кнопка «Создать задачу» (полная ширина, синяя)

Закрытие: крестик, клик по backdrop, или Escape.

---

## 5. Мобильная навигация

Нижняя панель (`BottomNav`) **удаляется полностью**.  
Кнопки «Проекты» и «Задачи» убираются.  
Единственный постоянный элемент навигации — FAB с `+` внизу справа.

---

## 6. Что не меняется

- Drag-and-drop (dnd-kit) — логика остаётся, только меняется иконка handle
- SwipeableRow на мобильном — остаётся как запасное действие
- Структура данных и API-роуты — не трогаются
- Страницы login / register — оставляем (там и так минималистично)
- Тёмная тема — не реализуется в этом редизайне
